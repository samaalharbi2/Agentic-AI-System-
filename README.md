# Building Agnetic AI Systems

# 📌 Overview

This project was developed as part of the training program by Saudi Data and AI Authority (SDAIA).

The project demonstrates how to build an intelligent agent using LLMs that can perform multi-step tasks, maintain state, and interact with external tools and data sources.

The system is designed using advanced frameworks such as LangChain and LangGraph, following best practices for scalability and production readiness.

# 🎓 Training Program

Building Agentic AI Systems

This program aims to equip participants with the skills to design and develop LLM-powered intelligent systems capable of:

Managing state
Executing multi-step workflows
Integrating with external tools and data sources

# 🚀 Features
Multi-step reasoning agent
State management using LangGraph
Integration with external tools / APIs
Modular and extensible architecture
LLM-powered decision making
